# EEPIP MVP Frontend

Education Easy-Pay Investment Plan - Binary MLM System Demo

## Overview

This is a **demonstration MVP** of the EEPIP (Education Easy-Pay Investment Plan) frontend interface. It showcases the key user interfaces for the binary MLM system that combines education investment with network marketing commissions.

## Features Implemented

### 1. **Public Landing Page**
- Hero section with value proposition
- Investment calculator
- Feature highlights
- How it works section
- Commission structure overview
- Partnership information
- Call-to-action sections

### 2. **AI Engineer Dashboard**
- Personal statistics (rank, team size, earnings)
- Investment status tracking
- Recent commissions display
- Rank progress tracker
- Quick action buttons

### 3. **Binary Tree Visualization**
- Interactive tree structure
- Node status indicators (active, pending, inactive)
- Member details on click
- Tree statistics
- Legend and controls

### 4. **Enrollment/Registration System**
- Multi-step registration form
- AI Engineer information
- Student nomination details
- Investment configuration
- Review and submit process

### 5. **Commission Reports**
- Overview with breakdown charts
- Detailed transaction history
- Binary pairing analysis
- Matching bonus tracking
- Export functionality

## Business Logic Demonstrated

### MLM Commission Structure
- **Direct Sponsor**: 2% immediate bonus
- **Binary Pairing**: 5% pool-based with pro-rating
- **Matching Bonus**: 1.5% across 3 generations
- **Leadership Pool**: 1.5% rank-based sharing

### Investment Model
- **Investment**: Rs. 1,800,000 LKR
- **Interest**: 16% p.a. (Rs. 24,000/month)
- **Education**: BSc + MPhil programs
- **Duration**: 3 years + additional MPhil time

### Rank System
- Starter → Bronze → Silver → Gold → Platinum → Diamond
- Progressive requirements and rewards
- Permanent rank achievement
